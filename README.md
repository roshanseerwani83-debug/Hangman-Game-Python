# Hangman Game

## Overview

This project is a text-based Hangman Game developed using Python. The player must guess a randomly selected word one letter at a time before running out of attempts.

## Features

- Random word selection
- Letter-by-letter guessing
- Attempt tracking
- Win condition
- Game Over condition
- Interactive console gameplay

## Technologies Used

- Python

## Test Scenarios

- Correct letter guesses
- Incorrect letter guesses
- Winning the game
- Running out of attempts

## Concepts Used

- Random Module
- Lists
- Strings
- Loops
- Conditional Statements
- User Input Handling
